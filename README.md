# php-todoApplication
## ❓ What is todoApplication?
이 어플리케이션은 php로 구현한 todoApplication입니다.  
php 연습용으로 구현하였으며, 유저는 오늘의 할 일을 기본적으로 조회/작성/수정/삭제가 가능합니다.

## 💻 About this project
- IDE : NOVA
- OS : MacOS
- Database : MySQL
- Back-end : PHP, AWS  
- View engine : php
- configuration management tool : git
- Tools : Sequel Ace

## 🤜🏻 Focus On
- 다양한 함수들의 주석처리
- php의 GET/POST 방식 구현
- Database와의 연결 보안 부분
- 예외처리
